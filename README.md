# EV Trip Planner

面向特斯拉及其他电动车用户的旅行规划 Web App，移动端优先。支持路线规划、每日行程、地图点位、费用预估，以及「我的行程」管理。

## 功能

- **首页**：输入出发地、目的地、天数、车型，生成行程
- **行程详情**：总览卡片、每日行程（可折叠）、加入我的行程、查看地图、费用
- **行程地图**：按天筛选、按类型筛选（充电/美食/景点/酒店）、地图/列表视图、点位详情
- **费用页**：总费用、明细列表、占比、预算建议
- **我的行程**：保存的行程列表、状态筛选、查看详情/地图/删除
- **个人中心**：头像与昵称、我的行程入口、主题切换、登录/注册/退出
- **登录/注册**：表单登录注册、第三方预留、游客入口
- **主题**：浅色 / 深色切换，本地持久化

## 技术栈

- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- 前端演示版，mock 数据，结构预留 API 对接

## 本地运行

```bash
# 安装依赖（若 npm 报权限错误，可先执行：sudo chown -R $(whoami) ~/.npm）
npm install

# 开发
npm run dev

# 构建
npm run build

# 生产预览
npm run start
```

浏览器打开 http://localhost:3000 。

## 项目结构

```
src/
  app/           # 页面（首页、登录、注册、行程详情、地图、费用、我的行程、个人中心）
  components/    # 通用与业务组件
  context/       # ThemeContext、AppContext
  lib/           # types、mockData
```
